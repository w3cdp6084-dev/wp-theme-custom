== Description ==
**Ultimate Preloader** will create a preloading screen for your website before all your images (including the images in CSS) are fully loaded. 

It will come in handy when you wish to preload all the images on your page before exposed to user.  
You can easily customize *Ultimate Preloader* as you wish.

Demo 1: http://www.2klabs.com/updemo/demo1/
Demo 2: http://www.2klabs.com/updemo/demo2/

== Installation ==
1. Upload `ultimate_preloader` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings -> Ultimate Preloader' in Wordpress menu to configure plugin
